#  Financial Expense Tracking & Analytics System

A Streamlit-based application to track expenses, analyze spending patterns, compare monthly budgets, predict future expenses, and send SMTP email alerts.


#  Features

- User Login & Registration
- Expense Tracking
- Budget Analysis
- Monthly Expense Comparison
- SMTP Email Notifications
- Future Expense Prediction
- Interactive Charts
- Download Reports

---

#  Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
- Scikit-learn
- SMTP
